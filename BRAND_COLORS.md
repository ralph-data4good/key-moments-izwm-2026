# IZWM 2026 Brand Colors Integration

## Official IZWM 2026 Color Palette

Based on the official IZWM branding guidelines, these colors differentiate the IZWM landing page from the Festival site:

### Primary Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Brick Red** | `#952925` | Week 2: Barriers to Zero Waste, primary CTA backgrounds |
| **Gold** | `#EDB849` | Week 4: False Solutions, accent highlights, gold elements |
| **Lime Green** | `#C2CB32` | Week 3: Solutions Week, buttons, positive actions |
| **Blue** | `#1586C0` | Week 5: Humans of Zero Waste, links, interactive elements |
| **Navy** | `#1A2D5E` | Week 1: Launch, hero gradients, deep backgrounds |

### Supporting Colors

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Cream** | `#F5F3ED` | Background sections, soft contrast |
| **Light Blue** | `#4a9fd4` | Hover states, secondary accents |
| **Mid Green** | `#7BA872` | Tertiary accents, in-person event chips |

## Visual Implementation

### Weekly Themes Timeline
- **Week 1**: Navy (`#1A2D5E`)
- **Week 2**: Brick Red (`#952925`)
- **Week 3**: Lime (`#C2CB32`)
- **Week 4**: Gold (`#EDB849`)
- **Week 5**: Blue (`#1586C0`)

The rainbow timeline gradient flows through all 5 IZWM brand colors.

### Sections Updated
1. ✅ **Hero Section** - Navy/Brick gradients
2. ✅ **Weekly Themes** - All 5 IZWM colors
3. ✅ **About Section** - Brick/Navy gradients, Blue/Lime accents
4. ✅ **Get Involved** - Mixed IZWM colors
5. ✅ **Buttons & CTAs** - Gold and Lime primary actions
6. ✅ **Links & Icons** - Blue for interactivity

### Event Card Weekly Theme Badges
- Each week's events display a badge in their corresponding weekly theme color
- Creates visual consistency between themes section and events

## Differentiation from Festival Site

### Festival Site Colors (for reference):
- Lime: `#A4BF3D` (different shade)
- Blue: `#4A8FC7` (lighter)
- Green: `#5B8C5A`
- Gold: `#D4A23D` (muted)

### IZWM 2026 Colors:
- Lime: `#C2CB32` (brighter, more vibrant)
- Blue: `#1586C0` (deeper, more professional)
- Brick Red: `#952925` (bold, distinctive)
- Gold: `#EDB849` (warmer, more inviting)
- Navy: `#1A2D5E` (strong, authoritative)

## Brand Consistency

All color variables are defined in `src/styles/tokens.css`:
- `--izwm-brick`, `--izwm-gold`, `--izwm-lime`, `--izwm-blue`, `--izwm-navy`
- Legacy variables (`--zw-*`, `--accent-*`) now map to IZWM colors
- Ensures backward compatibility while using new brand colors

- ✅ Professional, purposeful color choices
